# Mariam YSF 🌸

A beautiful, interactive React application for a special date request with animated flowers, confetti, and heartfelt messaging.

## Features

✨ Beautiful animated UI with floating flowers and confetti
💖 Interactive date request interface
🌷 Custom SVG flower animations
📱 Fully responsive design
🎨 Modern glassmorphism design
💬 Personalized messaging

## Tech Stack

- **React 18** - UI library
- **Vite** - Build tool
- **CSS** - Styling with animations

## Development

### Prerequisites
- Node.js (v16 or higher)
- npm or yarn

### Installation

```bash
npm install
```

### Running Locally

```bash
npm run dev
```

The application will start at `http://localhost:3000`

### Building for Production

```bash
npm run build
```

The production build will be in the `dist/` folder.

### Preview Production Build

```bash
npm run preview
```

## Deployment on Vercel

This project is configured for easy deployment on Vercel:

1. Push your code to GitHub
2. Import your repository in [Vercel Dashboard](https://vercel.com)
3. Vercel will automatically detect the Vite configuration and build settings
4. Your app will be deployed instantly!

### Environment

- **Build Command**: `npm run build`
- **Output Directory**: `dist`
- **Development Command**: `npm run dev`
